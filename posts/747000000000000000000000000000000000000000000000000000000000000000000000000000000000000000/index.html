安装 Oracle

1

 > docker run -d -it --name orcl19c_03 -p 1521:1521 -p 5500:5500 -e ORACLE_SID=orcl -e ORACLE_PDB=orclPDB1 -e ORACLE_PWD=123456 -e ORACLE_EDITION=standard -e ORACLE_CHARACTERSET=AL32UTF8 -v /data/oracle/oracleData:/opt/myData/oracle/oracleData registry.cn-hangzhou.aliyuncs.com/zhuyijun/oracle:19c

上面执行的命令中，备注：
ORACLE_SID=orcl 是服务名，安装后连接Oracle 时用的服务名，CDB数据库的
ORACLE_PWD=123456 是安装后 dba 用户的密码，即：123456
ORACLE_PDB=orclPDB1 是安装后的PDB数据库，想更明白一点，可以往下看6.3.1创建用户的两种方式，一个是在CDB库下，一个是下PDB库下

1

docker run -d -it --name orcl19c_03 -p 1521:1521 -p 5500:5500 -e ORACLE_SID=orcl -e ORACLE_PDB=orclPDB1 -e ORACLE_PWD=123456 -e ORACLE_EDITION=standard -e ORACLE_CHARACTERSET=AL32UTF8 -v /data/oracle/oracleData:/opt/myData/oracle/oracleData registry.cn-hangzhou.aliyuncs.com/zhuyijun/oracle:19c

Docker 常用命令

1. Docker 基础命令

  • 查看 Docker 版本：
1

docker --version
  • 启动 Docker 服务：
1

sudo systemctl start docker
  • 停止 Docker 服务：
1

sudo systemctl stop docker
  • 查看 Docker 服务状态：
1

sudo systemctl status docker
  • 启动 Docker 容器：
1

docker start <容器ID或名称>
  • 停止 Docker 容器：
1

docker stop <容器ID或名称>
  • 删除 Docker 容器：
1

docker rm <容器ID或名称>
  • 删除 Docker 镜像：
1

docker rmi <镜像ID或名称>
  • 列出所有正在运行的容器：
1

docker ps
  • 列出所有容器（包括停止的）：
1

docker ps -a
  • 列出所有镜像：
1

docker images

2. Docker 镜像操作命令

  • 拉取镜像：
1

docker pull <镜像名称>:<标签>
  • 构建镜像：
1

docker build -t <镜像名称>:<标签> .
  • 查看镜像详情：
1

docker inspect <镜像ID或名称>

3. Docker 容器操作命令

  • 运行容器：
1

docker run -d --name <容器名称> <镜像名称>
  • 进入正在运行的容器：
1

docker exec -it <容器ID或名称> /bin/bash
  • 导出容器为一个文件：
1

docker export <容器ID> > <文件名>.tar
  • 从文件中导入容器：
1

docker import <文件名>.tar

4. Docker 网络操作命令

  • 列出所有网络：
1

docker network ls
  • 创建自定义网络：
1

docker network create <网络名称>
  • 将容器连接到网络：
1

docker network connect <网络名称> <容器名称>

Docker 安装步骤

1. 安装前准备

  • 更新系统包：
1

sudo apt-get update

2. 安装 Docker

对于 Ubuntu 系统：
  1. 安装依赖：
1

sudo apt-get install apt-transport-https ca-certificates curl software-properties-common
  1. 添加 Docker 的 GPG 密钥：
1

curl -fsSL [https://download.docker.com/linux/ubuntu/gpg](https://download.docker.com/linux/ubuntu/gpg) | sudo apt-key add -
  1. 添加 Docker 的仓库：
1

sudo add-apt-repository "deb [arch=amd64] [https://download.docker.com/linux/ubuntu](https://download.docker.com/linux/ubuntu) $(lsb_release -cs) stable"
  1. 更新软件包并安装 Docker：
1
2

sudo apt-get update  
sudo apt-get install docker-ce
  1. 验证 Docker 安装：
1

sudo docker --version

3. 启动 Docker 并设置开机自启动

  • 启动 Docker：
1

sudo systemctl start docker
  • 设置 Docker 开机自启动：
1

sudo systemctl enable docker

4. 运行 Docker 测试

  • 运行一个简单的 Docker 容器来测试安装：
1

sudo docker run hello-world

这将下载并运行一个测试容器，确保 Docker 安装正确。

 配置 docker 镜像加速

编辑 daemon.json ：vim /etc/docker/daemon.json

添加下面的内容：

{
“registry-mirrors”: [
“https://mirror.ccs.tencentyun.com”
]
}

重载所有修改过的配置文件

systemctl daemon-reload

重启服务

systemctl restart docker
